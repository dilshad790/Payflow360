### Overview 
A Comprehensive Power BI Solution for Multi-Currency Financial Analysis

### Objective
Analyzed 12 months of UPI transaction data (Jan-Dec 2024) to identify payment trends and consumer behavior across 4 currencies.
### The Problem
Financial analysts often struggle to toggle between different visual perspectives and currencies without cluttering the report.

### 🧠 Core Analytics & Logic Deep-Dive
1. Dynamic User Experience (UX)

Bookmark Integration: Implemented 4 bookmark-driven buttons that allow users to toggle between Line and Column charts instantly. This reduces navigation steps by 50%.


Advanced Filtering: Engineered 10 cross-filtering slicers—including Bank Name, Device Type, and Purpose—to allow for hyper-granular data exploration.

### Business Impact & Insights

 - Seasonal Trends: Discovered dual transaction peaks in May (1,707K) and October (1,691K).


- Capacity Planning: Identified a 6.8% swing in transaction volume between peaks and troughs, providing a blueprint for proactive infrastructure planning.
